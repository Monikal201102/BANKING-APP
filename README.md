# Banking Application 💳

A full-stack MERN banking application with secure authentication and transaction management.

## 🚀 Features
- User Registration & Login (JWT Authentication)
- Deposit & Withdraw Money
- Transaction History Tracking
- Loan Recommendation System

## 🛠️ Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## 📂 Project Structure
- banking-frontend (React UI)
- banking-backend (Node.js API)

## ▶️ How to Run
1. Clone the repo
2. Install dependencies:
   npm install
3. Run backend and frontend:
   npm start
